# Meditation User Story

**Meditation App User Story**
As a person new to meditation I want to be able to find easy beginner techniques to get me started on my mediationa journey.

**Acceptance Criteria**
1. Able to find techniques and excersises easily
2. Able to favourite techniques and excersises
3. Able to share techniques with friends

**Story Points**

*User Story - Login/Registration*

- As a User, I want to register my account with my username, email, password
    Acceptance criteria: User enter valid details and click on register button to complete registration. If any details missing, show error message.
- As a User, I want to login to my account with my username and password
    Acceptance criteria: User enter valid details and click on login to sign in. If any details missing or incorrect, show error message.
- As a User, I want my data to be in local storage so that it persists during sessions
    Acceptance criteria: User details are stored in local storage and used for authentication in login process.

*User Story - Homepage*

- As a User, I want a greeting with my name, so that I know I have successfully logged in with my credentials.
    Acceptance criteria: Display greeting with username, Followed by the title "Started you meditation Journey here"
- As a USer, I want to see beginnger meditation cards, so that I can get started with the journey
    Acceptance criteria: Display cards with title, image, description and duration.
- As a User, I want to see a daily featured meditation excercise so that I can find new excercises to keep my motivation up.
    Acceptance criteria: Showcase one excersie/technique with title, image, description and duration.
- As a User, I want to see intiutive navigation icons so that I can easily move around the app.
    Acceptance criteria: A logo in the top left corner and a settings icon in the top right

*User Story - Detailed page*

- As a User, I want to see an about section for the excercise/technique
    Acceptance criteria: An about section explaining its focus and benefits
- As a User, I want to see an instructions section for the excerise/technique
    Acceptance criteria: Provide step-by-step guidance
- As a User, I want to a add to favorites button
    Acceptance criteria: Add to favorites button at the bottom of the page.
- As a User, I want a navigation button to go back to homepage and a button to share the excercie/technique with a friend.

*User Story - Favorites*
- As a User, I want to favorite (heart filled with red color) and unfavorite (heart with no fill color) excersies.
    Acceptance criteria: A heart icon with the text add to favorites near each card.
- As a USer, I want a favorites screen so that I can easily find my favorite excercises/techniques
    Acceptance criteria: A seperate screen with all favorites

*User Story - Favorites*
- As a User, I want to select dates for reminders, with a easy to navigate calendar.
    Accceptance criteria: An easy to navigate calender and able to click on a date to schedule reminders.
- As a User, I want to see a list of set reminders so that i can easily view them.
    Acceptance criteria: A button on the homepage that navigates to the reminders list, and a button near each set reminder to remove it if the user wants.

*User Story - Log out*
- As a User, I want to be able to log out after completing the session
    Acceptance criteria: go to setting screen and click on the logout button to log out and be redirected to login screen.

*User Story - Change Setting*
- As a User, I want to be able to change the theme (light mode and dark mode)
    Acceptance criteria: In the setting page have a button to toggle between light and dark mode.
